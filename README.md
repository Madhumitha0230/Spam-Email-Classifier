# 📧 Spam Email Classifier using Machine Learning

## 📌 Project Overview
This project is a Spam Email Classifier Web Application built using Machine Learning and Streamlit.  
It predicts whether a given message is **Spam** or **Not Spam (Ham)**.

The model is trained using Natural Language Processing (NLP) techniques and the Naive Bayes algorithm.

## ⚙️ How It Works

1. User enters a message.
2. Text is cleaned (lowercase, punctuation removal, stopwords removal).
3. Text is converted into numerical features using CountVectorizer.
4. The trained Multinomial Naive Bayes model predicts:
   - 🚨 Spam
   - ✅ Not Spam
